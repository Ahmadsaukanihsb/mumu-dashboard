Setup Termux (Redfinger)

1
Install Termux

Download dari F-Droid (bukan Play Store)

Download Termux
2
Buka Termux & Jalankan Setup

Copy-paste command ini di Termux:

pkg update -y && pkg install python adb git curl cloudflared -y && pip install flask flask-cors werkzeug && git clone https://github.com/Ahmadsaukanihsb/mumu-dashboard.git ~/dashboard-roblox && bash ~/dashboard-roblox/setup_termux.sh
3
Start Dashboard

Jalankan dashboard di Termux:

cd ~/dashboard-roblox && bash run_termux.sh
4
Start Remote Monitor (Optional)

Auto-rejoin & monitor untuk Cloudphone accounts:

cd ~/dashboard-roblox && python remote_monitor.py --url https://dashboard.aavpanel.my.id
5
Akses dari PC

Buka browser di PC:

https://dashboard.aavpanel.my.id

Remote Monitor Info

Yang Dilakukan

  • ✓ Auto-discover Roblox packages
  • ✓ Auto-create accounts di dashboard
  • ✓ Auto-extract cookies
  • ✓ Auto-push scripts ke apps
  • ✓ Auto-rejoin saat kicked/disconnect
  • ✓ Monitor status ke dashboard

Persyaratan

  • ✓ Root access di device
  • ✓ Internet connection
  • ✓ Roblox apps terinstall
  • ✓ Delta Executor terinstall
  • ✓ Dashboard PC running

Commands Reference

Setup

# Install dependencies
pkg update -y
pkg install python adb git curl cloudflared

# Install Python packages
pip install flask flask-cors werkzeug

# Clone project
git clone https://github.com/Ahmadsaukanihsb/mumu-dashboard.git ~/dashboard-roblox

# Setup
bash ~/dashboard-roblox/setup_termux.sh

Run

# Start dashboard
cd ~/dashboard-roblox
bash run_termux.sh

# Start remote monitor
python remote_monitor.py --url https://dashboard.aavpanel.my.id

# Update project
cd ~/dashboard-roblox
git pull
